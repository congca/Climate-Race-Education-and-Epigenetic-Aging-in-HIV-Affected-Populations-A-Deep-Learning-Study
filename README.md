# Climate, Air Pollution, and Epigenetic Aging in HIV

Analysis code for the manuscript:

Social Determinants Outweigh Climate Factors in Epigenetic Aging Among People Living With HIV

## Files

 `Data_Preparation_stage1_clean.py` – data preprocessing
 `Random_Forest_Feature_Selection_stage1_clean.py` – random forest analyses
 `K_Means_and_DAG_stage1_clean.py` – clustering and DAG analyses
 `DL_prediction_stage1_clean.py` – deep learning models
 `GENE_stage1_clean.py` – gene analyses
 `NEW LM MODEL -1.R` – linear regression analyses
 `NEW LM model-2.R` – interaction models
 `data_clean_stage1.R` – data cleaning

## Data

The study uses MWCCS/MACS data together with publicly available air pollution and meteorological data. Individual-level participant data are not included in this repository.

## Running the code

The scripts were developed independently during different stages of the project. Update file paths before running them.
